# CONTENT_RULES.md — 英文内容与文案规则

## 1. 品牌语气

PostCraft.work 的语气：

- Clear
- Professional
- Practical
- Calm
- Helpful
- Tool-first

不要：

- 夸张营销
- AI 味
- 空泛口号
- 过度承诺
- 冒充 LinkedIn 官方
- 强行说自己 best / #1

## 2. 核心文案

Brand line：

```txt
Free LinkedIn Post Tools
```

Main subtitle：

```txt
Format, preview, inspect, and polish your LinkedIn posts before publishing.
```

Trust note：

```txt
No sign-up required · Browser-based · Copy-ready formatting
```

Independent disclaimer：

```txt
PostCraft.work is an independent tool and is not affiliated with LinkedIn.
```

## 3. Button 文案

Primary：

```txt
Format post
Copy post
Preview post
Check post
Convert text
Copy bold text
```

Secondary：

```txt
Clean spacing
Clear
Use example
Add bullets
Copy plain text
```

不要使用模糊按钮：

```txt
Submit
Go
Magic
Generate
Boost
Optimize everything
```

## 4. Inspector 文案

Inspector 不要说：

```txt
AI score
Viral score
Guaranteed engagement
Best post quality
```

可以说：

```txt
Publishing readiness
Readability checks
Structure checks
Formatting checks
```

示例：

Good：

```txt
Your post has readable paragraph spacing.
```

Warning：

```txt
Your opening section is a little long. Consider making the first idea easier to scan.
```

Fix：

```txt
Add paragraph breaks so your post is easier to read on mobile.
```

## 5. Formatter 文案

要强调“不改写内容，只整理格式”：

```txt
Format your post without rewriting your message.
```

```txt
Clean up spacing, paragraph breaks, and bullets before copying your post into LinkedIn.
```

## 6. Preview 文案

不要承诺 100% 一致：

```txt
This preview is an approximate feed-style preview. LinkedIn may render posts differently depending on device, app version, and account settings.
```

## 7. Bold Text 文案

必须说明 Unicode：

```txt
LinkedIn posts do not provide native rich text formatting for regular posts. This tool uses Unicode characters to simulate bold or italic text.
```

辅助说明：

```txt
Use Unicode bold text sparingly. Some characters may be harder to read with assistive technologies.
```

## 8. 隐私文案

因为工具本地处理：

```txt
Your text is processed in your browser and is not uploaded to a server.
```

不要写不能保证的绝对承诺，比如：

```txt
100% private forever
military-grade privacy
```

## 9. FAQ 风格

FAQ 答案控制在 2-5 句。

示例：

Q：

```txt
Is PostCraft.work affiliated with LinkedIn?
```

A：

```txt
No. PostCraft.work is an independent tool. It helps you format and preview text before copying it into LinkedIn, but it is not affiliated with LinkedIn.
```

Q：

```txt
Does this tool post directly to LinkedIn?
```

A：

```txt
No. It only helps you prepare text. You copy the result and paste it into LinkedIn yourself.
```

Q：

```txt
Is my post uploaded?
```

A：

```txt
No. The formatting, preview, and checks run in your browser.
```

## 10. SEO 内容写法

每个 SEO section：

- 先回答用户问题。
- 再说明工具怎么解决。
- 最后给 2-4 条实用 tips。

不要写废话：

```txt
In today's digital world, LinkedIn is very important...
```

要直接：

```txt
A readable LinkedIn post usually has a clear opening line, short paragraphs, and a simple call to action.
```

## 11. 避免商标风险

可以写：

```txt
LinkedIn post formatter
```

不要写：

```txt
Official LinkedIn formatter
LinkedIn-approved tool
LinkedIn partner tool
```
